Media Art

ドーパミン中毒の可視化・身体化

Demo Video
Description

現代社会において、TikTok や YouTube Shorts に代表される縦型ショート動画プラットフォームは、我々の情報摂取のスタイルを劇的に変容させた。スマートフォンの画面を指先一つでスワイプするだけで、ユーザーの嗜好に最適化されたコンテンツが次々と供給される体験は、かつてない利便性と快楽をもたらしている。

しかしこの摩擦の極めて少ないインターフェースと、高度に洗練されたレコメンドアルゴリズムの結合は、ユーザーから情報の選択権を静かに奪いつつある。次々と現れる動画への期待感は、脳内の報酬系を刺激し、気づけば数時間を浪費してしまうというドーパミン中毒とも呼べる依存状態を引き起こしている。私たちはテクノロジーを道具として使用しているつもりで、実のところシステムによって時間を消費させられているのではないか。この主客の逆転は、現代人が直面している看過できない課題である。

この依存構造を助長している最大の要因は、操作に伴う身体的コストの欠如にあると本研究は考える。スワイプという動作は肉体的な負荷がほぼゼロであるため、脳が情報の過剰摂取によって疲弊していても、身体はその警告（疲れ）を発することができない。身体的な限界というブレーキを失った状態では、無限に続くアルゴリズムのループから自律的に脱出することは極めて困難である。テクノロジーの進化がインターフェースから物理的な重みを取り除いてきた結果、人間は自らの行動を制御するための重要な身体感覚を喪失してしまったと言える。

本研究の目的は、この不可視化されたアルゴリズムによる支配構造を、物理的な装置を通じて可視化し、鑑賞者に批判的な気づき（リテラシー）を与えることにある。そのためのアプローチとして、本研究では日本の伝統的な抽選器である「福引き」をインターフェースに採用した体験型作品を制作した。本来、運試しのために回す福引きを、情報の獲得手段として再定義し、鑑賞者に面白い映像（報酬）を得るための労働として、重いハンドルを回し続けさせる。
